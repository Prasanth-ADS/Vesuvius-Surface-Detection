# üìå Notes about this requirements list

### ‚≠ê `tifffile` + `imagecodecs`

These two packages **fix the exact LZW decoding error** you had earlier:

```
ValueError: <COMPRESSION.LZW: 5> requires the 'imagecodecs' package
```

Without `imagecodecs`, Vesuvius datasets fail to load on Windows/Linux.
